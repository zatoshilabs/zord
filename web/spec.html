Global / Blockchain

  • GET /api/v1/status – index height, chain tip, components
  • GET /block/height – latest indexed height
  • GET /api/v1/inscriptions?page=&limit= – recent inscriptions
  • GET /inscription/:id (HTML), /preview/:id, /content/:id

ZRC‑20

  • GET /api/v1/tokens?page=&limit=&q= – token list
  • GET /api/v1/zrc20/token/:tick – token info (base units)
  • GET /api/v1/zrc20/token/:tick/balances?page=&limit= – holders
  • GET /api/v1/zrc20/address/:address – address portfolio
  • GET /api/v1/zrc20/transfer/:id – staged transfer (with used and outpoint)
  • GET /api/v1/zrc20/token/:tick/integrity – supply vs holders sum
  • GET /api/v1/zrc20/status – module height
  • Compat: /token/:tick, /token/:tick/balance/:address
